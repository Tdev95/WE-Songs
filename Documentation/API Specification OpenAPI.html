Get some artists (artistsGet)
Returns a list of all artists in the dataset, optionally filtered by artist name and/or genre

Query Parameter — The name of the artist
genre (optional)
Query Parameter — The genre of the song

Query Parameter — Sort artists by hotttnesss

array[Artist]
